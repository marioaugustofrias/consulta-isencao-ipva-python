# 🚗 Consulta de Isenção de IPVA

Projeto desenvolvido em Python para calcular se um veículo ainda precisa pagar IPVA ou se já atingiu o tempo necessário para possível isenção.

## 📌 Sobre o projeto

Este projeto surgiu da ideia de transformar uma dúvida comum em uma ferramenta simples e automatizada.

O usuário informa o ano do veículo e o sistema calcula:

* quantos anos o carro possui;
* se ainda existe cobrança de IPVA;
* quanto tempo falta para atingir a faixa de isenção.

Além disso, o sistema permite realizar novas consultas sem precisar reiniciar o programa.

---

## ⚙️ Tecnologias utilizadas

## ✨ Funcionalidades

✅ Consulta baseada no ano do veículo
✅ Cálculo automático da idade do carro
✅ Validação para anos inválidos
✅ Opção de realizar múltiplas consultas

---

## ▶️ Como executar

1. Instale o Python
2. Baixe o arquivo do projeto
3. Execute no terminal:

```bash
python consulta_ipva.py
```
